# wpath
for Subsystem linux using users on Windows 10 
Drop files path change your linux system path
Example 
┌─[laperis@DESKTOP-LCRQVL5]─[~]
└──╼ $wpath 'C:\Users\maaam\Desktop\NM07.pcapng'
/mnt/c/users/maaam/desktop/nm07.pcapng
